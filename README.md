# 🚀 Crypto Tracker

A modern cryptocurrency tracking web application built with React and Vite. The application provides real-time cryptocurrency market data, interactive price charts, and detailed information about digital assets through a clean and responsive user interface.

## ⚙️ Tech Stack

- React 19
- Vite
- React Router
- Recharts
- CoinGecko API
- Context API
- Custom Hooks
- CSS3

## ✨ Features

### 🪙 Real-Time Cryptocurrency Tracking
- Live cryptocurrency prices
- Real-time market updates
- Top cryptocurrencies overview

### 📈 Interactive Price Charts
- 7-day price history visualization
- Responsive charts powered by Recharts
- Easy trend analysis

### 🔍 Detailed Coin Information
- Individual coin detail pages
- Market capitalization data
- Trading volume statistics
- Price change indicators
- Cryptocurrency rankings

### 🎯 Modern Navigation
- Dynamic routing with React Router
- Smooth page transitions
- User-friendly navigation experience

### 📱 Responsive Design
- Mobile-friendly interface
- Optimized for desktop, tablet, and mobile devices
- Modern UI with smooth animations

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Aahan2214/REACT-PROJECTS.git
```

### Navigate to the Project

```bash
cd REACT-PROJECTS
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
